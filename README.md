# 🛡️ Vulnerability Proof-of-Concept (PoC) Repository

This repository contains curated Proof-of-Concept (PoC) exploits for publicly disclosed vulnerabilities (CVEs). 

The goal is to provide a clean, scalable, and searchable structure for security research, learning, and testing.


> NOTE: Some PoCs are referenced from external sources and may not be fully validated. Always review and test responsibly.


## 🌐 Web UI

Features:

- Search by CVE/vendor/product/category/tags
- Filter by severity and category
- Full detail panel with metadata, references, and README contents (when available by CVE path)
- Automated indexing and updates from `manifest.json`
- CVSS integration
- Theme toggle
